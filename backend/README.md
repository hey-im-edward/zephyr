# Backend

Spring Boot 3.5.12 REST API for the shoe-store migration.

## Stack

- Java 21
- Spring Web MVC
- Spring Data JPA
- Spring Security
- Flyway
- MySQL

## Run

```bash
./mvnw spring-boot:run
```

## Environment

Copy values from `.env.example` into your shell or local environment:

- `SPRING_DATASOURCE_URL`
- `SPRING_DATASOURCE_USERNAME`
- `SPRING_DATASOURCE_PASSWORD`
- `APP_ADMIN_USERNAME`
- `APP_ADMIN_PASSWORD`
- `APP_CORS_ALLOWED_ORIGINS`
